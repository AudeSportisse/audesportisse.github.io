3.1. Retrouver un nombre mystère

Question 1

On choisit par exemple 7 comme nombre mystère à retrouver.

n <- 10
myst <- 7
tirage <- n+1
nbtirages <- 0

while(tirage!=myst){
  tirage <- sample(0:n,size = 1)
  nbtirages <- nbtirages+1
}

nbtirages
## [1] 1

Question 2

La fonction s’en déduit sans problème.

f <- function(n,myst){
  tirage <- n+1
  nbtirages <- 0
  while(tirage != myst){
    tirage <- sample(0:n, size = 1)
    nbtirages <- nbtirages+1
  }
  return(nbtirages)
}

Question 3

On applique \(M\) fois cette fonction.

n <- 10
myst <- 7
M <- 100
res <- rep(0,M)
for (m in 1:M){res[m] <- f(n,myst)}

La moyenne peut s’obtenir comme suit :

mean(res)
## [1] 14.31

Pour l’écart-type :

sd(res)
## [1] 15.66292

Théoriquement, le nombre de tirages \(X\) suit une loi géométrique de paramètre \(1/11\), donc de moyenne \(\mathbb{E}[X]=11\) et d’écart-type \(\sigma=\sqrt{110}\approx 10.5\). Pour rappel, la loi géométrique permet de modéliser une suite finie d’expériences aléatoires indépendentes deux à deux, chacune ayant deux issues: un succès (ici: le nombre obtenu par tirage est 7) ou un échec (ici: le nombre obtenu par tirage n’est pas 7). La loi géométrique est la loi du nombre d’expériences nécesssaires pour obtenir le premier succès (ici: nombre de tirage avant d’obtenir 7 pour la première fois).

3.2. La conjecture de Syracuse

Question 1

syracuse <- function(n){
  suite <- n
  l <- 1
  while(suite[l] != 1){
    if (suite[l]%%2 == 0){suite <- cbind(suite, suite[l]/2)
    }else{suite <- cbind(suite, 3*suite[l]+1)}
    l <- l+1
  }
  return(suite)
}

Question 2

On applique cette fonction à tous les entiers de 1 à 100.

N <- 100
longueur <- rep(0,N)
for(n in 1:N){
    longueur[n] <- length(syracuse(n))
}
plot(longueur,pch = 3)

Question 3

La longueur maximale est :

## [1] 119

## [1] 97

Pour représenter cette suite :

syrmax <- syracuse(which.max(longueur))

    xlab = '',ylab = '', main = 'Suite de Syracuse pour n=97'
    )
